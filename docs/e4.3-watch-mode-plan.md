## Open Questions / Risks

- **`--once` flag**: `infergen watch --once` runs exactly one scan+generate cycle then exits. This is the primary CI-testable surface; the live watch loop is verified by the `#[ignore]` test.
- **Debounce**: 300 ms after last relevant event before triggering re-cycle. Prevents rapid re-scans during a multi-file save.
- **notify crate version**: `notify = "6"` — stable, widely used, async-free channel API.
- **SIGINT**: Rust default signal handling terminates the process on Ctrl+C. No custom signal handler needed.
- **Initial generate on fresh dir**: `generate::run()` tolerates missing catalog (empty catalog → SDK preamble only). `scan::run()` creates the catalog. So the cycle always succeeds on a fresh dir.

**`run(args: WatchArgs) -> anyhow::Result<()>`** — entry point:

```rust
pub fn run(args: WatchArgs) -> anyhow::Result<()> {
    let cwd = std::env::current_dir()?;

    println!("watch: starting — initial scan");
    run_cycle(&cwd, &args.output)?;

    if args.once {
        return Ok(());
    }

    println!("watch: watching for changes (Ctrl+C to stop)");

    let (tx, rx) = mpsc::channel::<notify::Result<notify::Event>>();
    let mut watcher = notify::RecommendedWatcher::new(tx, notify::Config::default())?;
    watcher.watch(&cwd, notify::RecursiveMode::Recursive)?;

    let debounce = Duration::from_millis(300);
    let mut last_change: Option<Instant> = None;

    loop {
        match rx.recv_timeout(Duration::from_millis(50)) {
            Ok(Ok(event)) => {
                if event.paths.iter().any(|p| is_source_file(p)) {
                    last_change = Some(Instant::now());
                }
            }
            Ok(Err(e)) => eprintln!("watch: fs error: {e}"),
            Err(mpsc::RecvTimeoutError::Timeout) => {}
            Err(mpsc::RecvTimeoutError::Disconnected) => break,
        }

        if let Some(t) = last_change {
            if t.elapsed() >= debounce {
                last_change = None;
                println!("watch: change detected — re-scanning");
                if let Err(e) = run_cycle(&cwd, &args.output) {
                    eprintln!("watch: cycle error: {e}");
                }
            }
        }
    }

    Ok(())
}
```

**`run_cycle(cwd: &Path, output: &Path) -> anyhow::Result<()>`** — one scan + generate pass:

```rust
fn run_cycle(cwd: &Path, output: &Path) -> anyhow::Result<()> {
    let config = match infergen_core::Config::load_from_dir(cwd) {
        Ok(c) => c,
        Err(infergen_core::Error::ConfigNotFound { .. }) => Config::default(),
        Err(e) => return Err(e.into()),
    };

    // Scan
    let source_files = collect_source_files(cwd);
    println!("watch: {} source file(s)", source_files.len());
    let parser = JsParser;
    let mut all_proposals = Vec::new();
    for file_path in &source_files {
        let Ok(source) = std::fs::read_to_string(file_path) else { continue };
        let Ok(parsed) = parser.parse(file_path, &source) else { continue };
        let adapter = NextjsAdapter::new(cwd);
        all_proposals.extend(adapter.analyze(&parsed));
    }

    let catalog_path = cwd.join(&config.catalog);
    let merged = match load_catalog(&catalog_path) {
        Ok(existing) => rescan_merge(&existing, &all_proposals, cwd),
        Err(infergen_core::Error::Io(_)) => from_proposals(&all_proposals, cwd),
        Err(e) => return Err(e.into()),
    };
    save_catalog(&merged, &catalog_path)?;
    println!("watch: catalog saved ({} events)", merged.events.len());

    // Generate
    let ts = generate_typescript(&merged, &CodegenConfig::default());
    if let Some(parent) = output.parent()
        && !parent.as_os_str().is_empty()
        && !parent.exists()
    {
        std::fs::create_dir_all(parent)?;
    }
    std::fs::write(output, &ts)?;
    println!("watch: SDK written to {}", output.display());

    Ok(())
}
```

**`is_source_file(path: &Path) -> bool`** — filter for relevant TS/JS files, excluding ignored dirs:

```rust
fn is_source_file(path: &Path) -> bool {
    let has_source_ext = matches!(
        path.extension().and_then(|e| e.to_str()),
        Some("ts" | "tsx" | "js" | "jsx")
    );
    let in_ignored = path.components().any(|c| {
        matches!(
            c.as_os_str().to_str(),
            Some("node_modules" | ".git" | "dist" | "target" | ".next")
        )
    });
    has_source_ext && !in_ignored
}
```

**`collect_source_files` / `collect_files_recursive`:** Private copies of the helpers from `scan.rs` (same implementation, same ignore list).

### Edge cases

| Scenario | Behaviour |
|---|---|
| `--once` flag | Run initial cycle, exit 0 (no watcher setup) |
| No catalog yet (fresh project) | `from_proposals` creates one; generate writes empty SDK preamble |
| Cycle error (e.g., disk full) | `eprintln!` error, continue watching (non-fatal) |
| Non-source file changed | `is_source_file` returns false, debounce not triggered |
| node_modules file changed | `in_ignored` = true, filtered out |
| Watcher channel disconnected | `break` exits loop cleanly |
| SIGINT (Ctrl+C) | OS terminates process; no custom handler needed |
| Output parent dir missing | Created with `create_dir_all` before write |

/// Live watch test — spawns the watcher, creates a file, verifies re-scan fires.
/// Marked #[ignore] because it uses real file watching and timing.
/// Run manually: `cargo test -p infergen-cli watch_live -- --ignored`
#[test]
#[ignore]
fn watch_live_detects_new_source_file() {
    use std::time::Duration;
    use assert_cmd::cargo::cargo_bin;

    let dir = tempdir().unwrap();
    let mut child = std::process::Command::new(cargo_bin("infergen"))
        .args(["watch"])
        .current_dir(dir.path())
        .spawn()
        .unwrap();

    // Allow initial scan to complete
    std::thread::sleep(Duration::from_millis(800));

    // Create a Next.js page — watcher should detect and re-scan
    let pages_dir = dir.path().join("pages");
    std::fs::create_dir_all(&pages_dir).unwrap();
    std::fs::write(
        pages_dir.join("about.tsx"),
        "export default function AboutPage() { return null; }",
    ).unwrap();

    // Wait for debounce (300ms) + re-scan
    std::thread::sleep(Duration::from_millis(2000));
    child.kill().ok();

    let catalog = std::fs::read_to_string(dir.path().join(".infergen/catalog.yaml")).unwrap();
    assert!(catalog.contains("pageView"), "re-scan should detect about page event");
}
