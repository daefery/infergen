## Resolved decisions

1. **Core language = Rust.** Confirmed with user. Rationale: SWC/OXC (required by E0.3, the validation epic) are Rust crates embeddable as native dependencies; Rust gives the best single-binary distribution (PRD ¬ß7 Portability) and performance (PRD ¬ß7) for the scan-quality core bet. Python was considered and rejected ‚Äî it breaks the single-binary and performance NFRs. Go was rejected ‚Äî SWC/OXC would require cgo or a sidecar process.
2. **Rust crates via Cargo workspace; JS packages via pnpm workspace.** Two native workspaces, one repo. A top-level `Justfile` is the single cross-language task runner.
3. **Edition 2024, toolchain pinned `1.85.0`** (edition 2024 stabilized in Rust 1.85). Implementer may bump to the latest stable at build time; `1.85.0` is the floor.
4. **Initial version `0.0.0`** across all packages ‚Äî signals "nothing shipped yet," matching ROADMAP framing.
5. **Binary release automation via `cargo-dist` (the `dist` tool).** It is the modern standard for multi-platform single-binary Rust releases and directly seeds E8.3. JS package publishing is left **manual + publish-ready** for E0.1; full npm release automation is deferred to E8.3 (noted in `CONTRIBUTING.md`).
6. **`@infergen/*` npm scope; `infergen-*` Rust crate names.** Binary name = `infergen`.

```rust
//!
//! This crate will house the language parsers (E0.3), framework adapters
//! (E0.4), the heuristic namer (E1.2), and codegen (E2.x). For the E0.1

/// Errors produced by the scan engine.
///
/// Variants are added as subsystems land. `NotImplemented` is the placeholder
/// returned by stubs until their epic ships.
#[derive(Debug, thiserror::Error)]
#[non_exhaustive]
pub enum Error {
    /// A subsystem exists in the API but is not yet implemented.
    #[error("not yet implemented: {0}")]
    NotImplemented(&'static str),
}

/// Convenience result type for scan-engine fallible operations.
pub type Result<T> = std::result::Result<T, Error>;

/// The version string of the core engine, sourced from Cargo at compile time.
#[must_use]
pub fn version() -> &'static str {
    env!("CARGO_PKG_VERSION")
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn version_matches_cargo() {
        assert_eq!(version(), env!("CARGO_PKG_VERSION"));
    }

    #[test]
    fn not_implemented_error_formats() {
        let e = Error::NotImplemented("scan");
        assert_eq!(e.to_string(), "not yet implemented: scan");
    }

    #[test]
    fn reexports_schema_version() {
        assert_eq!(CATALOG_SCHEMA_VERSION, 1);
    }

```
