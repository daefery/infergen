## Recommendations

- E2.3 (Codegen Determinism & Safety) should add `infergen generate --check` that fails CI when the on-disk generated file differs from what would be generated from the current catalog.
- E3.1 (Provider Plugin Interface) will replace the `throw new Error("...")` stub in generated functions with real provider dispatch.
- The `track` object's `as const` enables TypeScript to narrow the call-site keys to exact event names — verify with a `.ts` type-check invocation in E2.3's golden-file test suite.
