---

## Overview

Adds five JS/TS framework adapters beyond the existing Next.js slice: **Express**, **NestJS**, **React Router**, **SvelteKit**, and **Vue/Nuxt**. Also adds minimal passthrough parsers for `.vue` and `.svelte` file types (so path-based detection works on these files) and two new `Language` enum variants.

All five adapters use line-based text scanning of `file.source` and/or path-based detection from `file.path` — consistent with the Go (E5.2) and Ruby (E5.3) adapters. No deep OXC AST traversal is required, which keeps adapters dependency-light and error-tolerant.

---

## Dependencies / Blocked-by

None. E0.4 is Done. All `Framework` enum variants (`React`, `Express`, `NestJs`, `Vue`, `SvelteKit`) already exist in `detect.rs`.

---

## Open Questions / Risks

- **`.vue` / `.svelte` parsing**: OXC/JsParser rejects these extensions. Solution: minimal `VueParser` / `SvelteParser` (source-passthrough, analogous to GoParser). Language variants `Language::Vue` / `Language::Svelte` added for clarity.
- **Express route detection false-positives**: Non-Express code may contain `.get('/path',` patterns. Mitigated by pre-filter: file source must contain `'express'` or `"express"`.
- **NestJS decorator nesting**: Only single-level `@Controller` + HTTP decorators detected in v1. Nested routes and `@ApiTags` not tracked.
- **Vue 3 Composition API routing**: `useRouter().push('/path')` is programmatic navigation, not a route definition. Not detected in v1 (low confidence anyway).
- **SvelteKit `+page.server.ts` vs `+page.svelte`**: Both map to the same route. Adapter emits one PageView per recognized file — dedup happens in catalog merge (E4.1).


**`extract_express_routes` logic:**
```rust
for line in source.lines() {
    let trimmed = line.trim();
    if trimmed.starts_with("//") { continue; }
    for method in EXPRESS_METHODS {
        let needle = format!(".{}(", method);
        if let Some(pos) = trimmed.find(needle.as_str()) {
            let after = &trimmed[pos + needle.len()..];
            if let Some(path) = extract_first_quoted(after) {
                if path.starts_with('/') {
                    routes.push((method.to_string(), path));
                }
            }
        }
    }
}
```

**Detection strategy — pure path-based:**

2. `rel = file.path.strip_prefix(&project_root)` — path relative to project root.
3. Must start with `src/routes/` or `routes/`.

5. **Classify by filename:**

   - `+layout.svelte`, `+error.svelte`, any other → skip
6. **Build route string from directory segments** (between `routes/` prefix and the filename):
   - Strip route groups: `(group_name)` → drop segment from path
   - Keep dynamic segments: `[param]` or `[...rest]` → include as-is in route name
   - Join remaining segments with `/`
   - Empty segments → route `"/"`

**Helper:** `route_from_sveltekit_path(rel: &Path) -> Option<(String, EventKind)>`

**`route_from_sveltekit_path` implementation:**
```rust
pub fn route_from_sveltekit_path(rel: &Path) -> Option<(String, EventKind)> {
    let comps: Vec<&str> = rel.components()
        .filter_map(|c| c.as_os_str().to_str())
        .collect();
    
    // Find "routes" root index
    let routes_idx = comps.iter().position(|&s| s == "routes")?;
    
    // Filename classification
    let filename = *comps.last()?;
    let kind = match filename {
        "+page.svelte" | "+page.server.ts" | "+page.server.js" => EventKind::PageView,
        "+server.ts" | "+server.js" => EventKind::ApiCall,
        _ => return None,
    };
    
    // Directory segments between "routes" and filename
    let dir_segs = &comps[routes_idx + 1..comps.len() - 1];
    let route_segs: Vec<&str> = dir_segs.iter()
        .filter_map(|seg| {
            // Drop route groups: "(marketing)" → None
            if seg.starts_with('(') && seg.ends_with(')') { return None; }
            Some(*seg)
        })
        .collect();
    
    let route = if route_segs.is_empty() {
        "/".to_owned()
    } else {
        format!("/{}", route_segs.join("/"))
    };
    
    Some((route, kind))
}
```

```rust
        .collect();

    if first != "pages" { return None; }
    
    let rest = &comps[1..];
    if rest.is_empty() { return None; }
    
    // Last component must be .vue file
    let last = *rest.last()?;
    if !last.ends_with(".vue") { return None; }
    
    let stem = last.strip_suffix(".vue")?;
    let segments: Vec<&str> = rest[..rest.len()-1].iter()
        .copied()
        .chain(if stem == "index" { None } else { Some(stem) })
        .collect();
    
    if segments.is_empty() {
        Some("/".to_owned())
    } else {
        Some(format!("/{}", segments.join("/")))
    }
}
```
