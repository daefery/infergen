## Overview

Adds monorepo awareness to Infergen: detect sub-packages inside a single repo root, namespace catalog events per package, merge per-package catalogs into a unified root-level view, and check cross-service event-name consistency.

PRD §10 specifies: "Lean: one catalog, namespaced by package." E5.5 implements exactly that:
- `detect_monorepo(root)` → `MonorepoLayout` (packages, their roots and detected stack)
- `package: Option<String>` field on `CatalogEntry` (backward-compatible schema extension)
- `namespace_catalog(catalog, package_name)` → tagged copy
- `merge_package_catalogs(packages)` → single merged, sorted catalog
- `cross_service_check(packages)` → `Vec<ConsistencyIssue>` (duplicate name, property conflict)

---

## Open Questions / Risks

- **Glob resolution**: npm workspace patterns like `packages/*` need glob expansion. Using simple prefix/wildcard logic (no external glob crate) keeps the dependency footprint small. Limitation: double-star globs (`apps/**`) not supported in v1.
- **Schema version**: `package` field added with `#[serde(default, skip_serializing_if = "Option::is_none")]`. Existing catalog.yaml files deserialize cleanly. `CATALOG_SCHEMA_VERSION` stays at 1.
- **Root package**: in npm/Cargo workspaces the root manifest coordinates sub-packages; the root itself is excluded from `packages` (it has no app code). Single-package projects return empty `packages` vec — callers use `is_monorepo()` to distinguish.
- **Name collision**: two packages with the same directory name → `name` field suffixed with index. Unlikely in practice but handled.

```rust
/// A single package discovered in a monorepo.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct MonorepoPackage {
    /// Human-readable package name (from manifest or directory name).
    pub name: String,
    /// Absolute path to this package's root directory.
    pub root: PathBuf,
    /// Default catalog path for this package: `{root}/.infergen/catalog.yaml`.
    pub catalog_path: PathBuf,
    /// Languages detected in this package (via [`detect::detect`]).
    pub languages: Vec<Language>,
    /// Frameworks detected in this package (via [`detect::detect`]).
    pub frameworks: Vec<Framework>,
}

/// The layout of a monorepo: root path and all discovered packages.
///
/// If `packages` is empty the root is a single-package project.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct MonorepoLayout {
    /// Absolute path to the monorepo root.
    pub root: PathBuf,
    /// Sub-packages discovered in the monorepo. Empty for single-package projects.
    pub packages: Vec<MonorepoPackage>,
}

impl MonorepoLayout {
    /// `true` when more than one package is present.
    #[must_use]
    pub fn is_monorepo(&self) -> bool {
        self.packages.len() > 1
    }

    /// Number of distinct languages across all packages (union).
    #[must_use]
    pub fn language_count(&self) -> usize {
        let mut langs: Vec<Language> = self.packages.iter()
            .flat_map(|p| p.languages.iter().copied())
            .collect();
        langs.sort_unstable_by_key(|l| *l as u8);
        langs.dedup();
        langs.len()
    }
}
```

#### `detect_monorepo`

```rust
/// Detect packages inside a monorepo root.
///
/// Scans for common workspace configurations:
/// 1. npm `package.json` `workspaces` field
/// 2. pnpm `pnpm-workspace.yaml` `packages` field
/// 3. Cargo `Cargo.toml` `[workspace] members`
/// 4. Multiple manifest files in immediate subdirectories
///    (`package.json`, `go.mod`, `pyproject.toml`, `Gemfile`, `Cargo.toml`)
///
/// Each discovered package directory is passed through [`detect::detect`]
/// to populate `languages` and `frameworks`.
///
/// # Errors
/// Returns [`Error::Io`] only if `root` does not exist or is not a directory.
/// Individual sub-directory detection failures are silently skipped.
pub fn detect_monorepo(root: &Path) -> Result<MonorepoLayout, Error>
```

**Internal helper: `detect_npm_workspace_dirs(root) -> Vec<PathBuf>`**
1. Read `{root}/package.json`. Parse JSON. Look for `workspaces` key (array of strings).
2. For each pattern: call `resolve_workspace_pattern(root, pattern)`.
3. Also check `{root}/pnpm-workspace.yaml`: parse YAML `packages:` key (list of strings), same resolution.
4. Union both sets. Deduplicate.

**Internal helper: `detect_cargo_workspace_dirs(root) -> Vec<PathBuf>`**
1. Read `{root}/Cargo.toml`. Look for `[workspace] members` array.
2. Use `toml::from_str::<toml::Value>()`. Navigate `workspace.members` key.
3. Each member is a relative path string → `root.join(member)`.
4. Only include paths that are existing directories.

**Internal helper: `detect_multi_manifest_dirs(root) -> Vec<PathBuf>`**
1. Read immediate subdirectories of `root` (`read_dir(root)` depth = 1).
2. For each subdir: check if any of `["package.json", "go.mod", "pyproject.toml", "Gemfile", "Cargo.toml"]` exist.
3. Collect subdirs that have at least one manifest (excluding root's own `Cargo.toml` to avoid self-inclusion).
4. Only meaningful when ≥ 2 subdirs qualify.

**Internal helper: `resolve_workspace_pattern(root, pattern) -> Vec<PathBuf>`**
```rust
fn resolve_workspace_pattern(root: &Path, pattern: &str) -> Vec<PathBuf> {
    // Strip leading "./" if present
    let pattern = pattern.trim_start_matches("./");
    
    if let Some(prefix) = pattern.strip_suffix("/*") {
        // e.g. "packages/*" → list all subdirs under packages/
        let dir = root.join(prefix);
        if !dir.is_dir() { return vec![]; }
        read_dir(&dir).ok().into_iter().flatten()
            .filter_map(|e| e.ok())
            .map(|e| e.path())
            .filter(|p| p.is_dir())
            .collect()
    } else if !pattern.contains('*') {
        // Literal path
        let path = root.join(pattern);
        if path.is_dir() { vec![path] } else { vec![] }
    } else {
        // Other glob patterns not supported in v1
        vec![]
    }
}
```

**Internal helper: `package_name_from_dir(root, dir) -> String`**
1. Try `{dir}/package.json` → `name` field (JSON string).
2. Try `{dir}/Cargo.toml` → `[package] name` (TOML string).
3. Fallback: `dir.file_name().to_str()` → directory name.
4. If name collision with existing package names, append `-2`, `-3`, etc.

**`detect_monorepo` algorithm:**
```rust
pub fn detect_monorepo(root: &Path) -> Result<MonorepoLayout, Error> {
    if !root.is_dir() {
        return Err(Error::Io(...));
    }
    
    let mut package_dirs: Vec<PathBuf> = Vec::new();
    
    // Priority 1: npm / pnpm workspace config
    let npm_dirs = detect_npm_workspace_dirs(root);
    
    // Priority 2: Cargo workspace members
    let cargo_dirs = detect_cargo_workspace_dirs(root);
    
    // Priority 3: multi-manifest scan (fallback for Go/Python/Ruby monorepos)
    // Only apply if no explicit workspace config was found.
    let manifest_dirs = if npm_dirs.is_empty() && cargo_dirs.is_empty() {
        detect_multi_manifest_dirs(root)
    } else {
        vec![]
    };
    
    // Union all dirs, maintaining declaration order, deduplicating by canonical path.
    let mut seen: HashSet<PathBuf> = HashSet::new();
    for dir in npm_dirs.into_iter().chain(cargo_dirs).chain(manifest_dirs) {
        if dir.is_dir() && seen.insert(dir.clone()) {
            package_dirs.push(dir);
        }
    }
    
    // Build MonorepoPackage for each directory.
    let mut used_names: HashMap<String, u32> = HashMap::new();
    let mut packages: Vec<MonorepoPackage> = Vec::new();
    
    for dir in package_dirs {
        let raw_name = package_name_from_dir(root, &dir);
        let count = used_names.entry(raw_name.clone()).or_insert(0);
        *count += 1;
        let name = if *count == 1 {
            raw_name
        } else {
            format!("{raw_name}-{count}")
        };
        
        let (languages, frameworks) = detect::detect(&dir)
            .map(|r| (r.languages, r.frameworks))
            .unwrap_or_default();
        
        let catalog_path = dir.join(".infergen").join("catalog.yaml");
        
        packages.push(MonorepoPackage { name, root: dir, catalog_path, languages, frameworks });
    }
    
    Ok(MonorepoLayout { root: root.to_owned(), packages })
}
```

#### `namespace_catalog`

```rust
/// Return a copy of `catalog` with every entry's `package` field set to `package_name`.
///
/// Does not modify the original. Entries that already have a `package` value
/// are overwritten.
#[must_use]
pub fn namespace_catalog(catalog: &Catalog, package_name: &str) -> Catalog {
    Catalog {
        schema_version: catalog.schema_version,
        events: catalog.events.iter().map(|e| CatalogEntry {
            package: Some(package_name.to_owned()),
            ..e.clone()
        }).collect(),
    }
}
```

#### `merge_package_catalogs`

```rust
/// Merge per-package catalogs into a single root-level catalog.
///
/// Each package's catalog is first namespaced with [`namespace_catalog`].
/// Events are then concatenated and sorted by `(package_name, id)` for stable,
/// grouped diffs — all events from `"frontend"` appear before `"backend"`, etc.
/// Events with the same `id` within the same package are deduplicated.
///
/// An empty input returns an empty default catalog.
#[must_use]
pub fn merge_package_catalogs(packages: &[(&str, &Catalog)]) -> Catalog {
    let mut events: Vec<CatalogEntry> = Vec::new();
    let mut seen: HashSet<(String, String)> = HashSet::new(); // (package, id)
    
    for (name, catalog) in packages {
        let namespaced = namespace_catalog(catalog, name);
        for entry in namespaced.events {
            let key = (name.to_string(), entry.id.clone());
            if seen.insert(key) {
                events.push(entry);
            }
        }
    }
    
    // Sort by (package, id) — None package sorts last.
    events.sort_by(|a, b| {
        let pa = a.package.as_deref().unwrap_or("");
        let pb = b.package.as_deref().unwrap_or("");
        pa.cmp(pb).then(a.id.cmp(&b.id))
    });
    
    Catalog { schema_version: CATALOG_SCHEMA_VERSION, events }
}
```

#### `ConsistencyIssue` and `cross_service_check`

```rust
/// Nature of a cross-service event-name inconsistency.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum ConsistencyKind {
    /// Same event name appears in multiple packages — may be an intentional
    /// shared event or an accidental duplicate.
    DuplicateName,
    /// Same event name appears in multiple packages with different property
    /// sets — almost certainly unintentional.
    PropertyConflict,
}

/// A cross-service event consistency issue found by [`cross_service_check`].
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct ConsistencyIssue {
    /// Event name that appears in more than one package.
    pub event_name: String,
    /// Packages where this event name is defined.
    pub packages: Vec<String>,
    /// Nature of the inconsistency.
    pub kind: ConsistencyKind,
}
```

```rust
/// Check cross-service event consistency across multiple package catalogs.
///
/// Finds event names that appear in two or more packages. For each duplicate:
/// - If all occurrences have identical property name sets → [`ConsistencyKind::DuplicateName`].
/// - If property sets differ → [`ConsistencyKind::PropertyConflict`].
///
/// Only non-`Ignored` entries are checked (ignored events are intentional suppressions).
/// Returns issues sorted by `event_name` for stable output.
#[must_use]
pub fn cross_service_check(packages: &[(&str, &Catalog)]) -> Vec<ConsistencyIssue>
```

**Algorithm:**
```rust
// Build: event_name → vec<(package_name, sorted_property_names)>
let mut map: HashMap<String, Vec<(String, Vec<String>)>> = HashMap::new();
for (pkg_name, catalog) in packages {
    for entry in &catalog.events {
        if entry.status == EventStatus::Ignored { continue; }
        let mut prop_names: Vec<String> = entry.properties.iter()
            .map(|p| p.name.clone())
            .collect();
        prop_names.sort_unstable();
        map.entry(entry.name.clone())
            .or_default()
            .push((pkg_name.to_string(), prop_names));
    }
}

// Emit issues for names present in 2+ packages.
let mut issues: Vec<ConsistencyIssue> = Vec::new();
for (event_name, occurrences) in &map {
    if occurrences.len() < 2 { continue; }
    let pkg_names: Vec<String> = occurrences.iter().map(|(n, _)| n.clone()).collect();
    let all_props: Vec<&Vec<String>> = occurrences.iter().map(|(_, p)| p).collect();
    let all_same = all_props.windows(2).all(|w| w[0] == w[1]);
    let kind = if all_same {
        ConsistencyKind::DuplicateName
    } else {
        ConsistencyKind::PropertyConflict
    };
    issues.push(ConsistencyIssue {
        event_name: event_name.clone(),
        packages: pkg_names,
        kind,
    });
}
issues.sort_by(|a, b| a.event_name.cmp(&b.event_name));
issues
```
