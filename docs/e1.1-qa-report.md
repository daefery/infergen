## Recommendations

- **serde_yaml deprecation:** `serde_yaml v0.9.34` is author-deprecated. Consider migrating to `serde_yml` (community fork) or `marked_yaml` in a future hardening epic (E8.x). No functional impact for E1.1.
- **Adapter attribution:** `EventProvenance.adapter` is currently empty string. E1.2 (Heuristic Namer) should add an `adapter: String` field to `ProposedEvent` and set it in `NextjsAdapter::analyze`.
- **Confidence round-trip:** `f64` confidence in `CatalogEntry` avoids precision drift but means `0.9_f32` serializes as `0.8999999761581421` in YAML. A custom serializer rounding to 4 decimal places would improve human-readability. Deferred to E1.2 or E8.x.
