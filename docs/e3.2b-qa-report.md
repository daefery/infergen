## Design Decisions Validated

1. **Driver-agnostic callbacks** ‚Äî DB adapters accept `execute`/`run` callbacks, compatible with any Node.js DB client. Zero generated-file npm dependencies. ‚úì
2. **Single-table schema** ‚Äî `infergen_events` table with JSONB/JSON/TEXT properties column. Per-event typed views for structured access. ‚úì
3. **`infergen schema` CLI command** ‚Äî reads catalog, outputs SQL to stdout or file. Deterministic. ‚úì
4. **Dynamic `import("node:fs")` in FileProvider** ‚Äî deferred to call time, won't break browser bundlers at parse time. ‚úì
5. **MySQL string concatenation** ‚Äî avoids Rust raw-string backtick escaping issues; resulting TypeScript is valid. ‚úì
6. **ConsoleProvider zero-dependency** ‚Äî works browser + Node.js with no imports. ‚úì
