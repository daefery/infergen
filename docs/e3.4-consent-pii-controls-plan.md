## Open Questions / Risks

- `_dispatch` is exported as a public symbol. Since the generated file is user-visible, we use a leading underscore naming convention (consistent with `_providers`) to signal internal use without true hiding.
- `RedactFn` returning `null` drops the property entirely; returning `undefined` also considered but `null` is more explicit in TS.
- `configureInfergen` references `setConsent` etc. which are defined later in the same file — valid in TypeScript because module-level function declarations are resolved before any call-site executes.
- Region routing filters providers by `id`; providers with no matching id in the route are silently excluded. This is intentional: the region route is an allowlist.

#### `_dispatch` internal dispatcher
```ts
/**
 * @internal
 * Central dispatch function — applies consent gate, opt-out, PII redaction,
 * and region routing before forwarding to providers.
 * All generated track functions call this instead of iterating _providers directly.
 */
export function _dispatch(
  eventName: string,
  properties: Record<string, unknown>,
): void {
  // Opt-out and consent gate
  if (_optedOut || _consentState === "denied") return;

  // PII redaction
  let payload: Record<string, unknown> = properties;
  if (_redactFn !== null) {
    payload = {};
    for (const [key, value] of Object.entries(properties)) {
      const result = _redactFn(key, value, eventName);
      if (result !== null) payload[key] = result;
    }
  }

  // Region routing: filter providers to those allowed for the current region
  let targets = _providers;
  if (_region !== null && _regionRoutes[_region] !== undefined) {
    const allowed = _regionRoutes[_region];
    targets = _providers.filter(p => allowed.includes(p.id));
  }

  targets.forEach(p => p.track(eventName, payload));
}
```

### Key Logic / Edge Cases

- **Consent "unknown"** is treated as permissive (dispatch proceeds). Only "denied" blocks. This matches common consent-library behavior where "not yet answered" ≠ "denied".
- **`_redactFn` returning `null`** drops the property (key absent from payload). Returning `undefined` is treated as a non-null value and is preserved — callers should explicitly return `null` to drop.
- **Region routing** is an allowlist: if a region is active but has no route configured, all providers receive the event (graceful degradation).
- **Empty `Object.entries`** (no properties): redaction loop is a no-op, `payload` stays `{}`.
