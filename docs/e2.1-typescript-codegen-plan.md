## File Specs

### 1. `crates/infergen-core/src/codegen/mod.rs`

**Purpose:** Translate an approved `Catalog` into a deterministic TypeScript source string.

**Exported symbols:**

```rust
#[derive(Debug, Clone, Default)]
pub struct CodegenConfig {
    pub include_proposed: bool,  // default false — Approved events only
}

pub fn generate_typescript(catalog: &Catalog, config: &CodegenConfig) -> String
```

**Private helpers:**

| Helper | Signature | Purpose |
|--------|-----------|---------|
| `to_pascal_case` | `(name: &str) -> String` | `"user_signed_in"` → `"UserSignedIn"` |
| `to_camel_case` | `(name: &str) -> String` | `"user_signed_in"` → `"userSignedIn"` |
| `ts_type` | `(prop_type: Option<&str>) -> &'static str` | `string/number/boolean` → TS; else `"unknown"` |
| `write_header` | `(out: &mut String)` | Auto-gen comment + versions; no timestamp |
| `build_event_name_type` | `(events: &[&CatalogEntry]) -> String` | `export type EventName = \| "..." \| "...";` or `= never` |
| `write_interface` | `(out: &mut String, entry: &CatalogEntry)` | `export interface XxxProperties { ... }` |
| `write_function` | `(out: &mut String, entry: &CatalogEntry)` | `export function trackXxx(properties: XxxProperties): void` |
| `write_track_object` | `(out: &mut String, events: &[&CatalogEntry])` | `export const track = { ... } as const;` |

**Key logic:**

- Filter: `Approved` always included; `Proposed` only if `config.include_proposed`; `Ignored` always excluded.
- Sort events alphabetically by name before output — ensures deterministic output.
- Name converters delegate to `crate::namer::split_identifier` for word-boundary splitting.
- Header contains no timestamp — output must be identical for same catalog across runs.
- Generated functions throw `Error("Infergen runtime not configured...")` — replaced by E3.1.

**Dependencies:** `infergen_types` (via `crate`), `crate::namer::split_identifier`

**Tests (unit — 29):**

| Group | Cases |
|-------|-------|
| `to_pascal_case` | single word, snake_case, single char |
| `to_camel_case` | single word, snake_case, single char |
| `ts_type` | string, number, boolean, None→unknown, unrecognised→unknown |
| `generate_typescript` | empty catalog (EventName=never, empty track), single event (interface, function, track key), PII tag present/absent, description in JSDoc, empty description → single-line JSDoc, Ignored excluded, Proposed excluded by default, Proposed included with flag, alphabetical sort, EventName union lists all, deterministic, empty interface for no-prop event, typed properties (string/number/boolean/unknown) |

**Score: 97%**
